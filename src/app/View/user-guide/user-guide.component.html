ဤ Timer ကို အသုံးပြုမည်နည်း?
            
  1. Add Task: လုပ်ဆောင်ရန်အလုပ် အားအရင် ထည့်ပါ
  2. Set estimate Time: (၁ = ၂၅ မိနစ်)အချိန်ခန့်မှန်းမှုအကြိမ် ကို တစ်ခုချင်းစီအတွက် သတ်မှတ်ပါ
  3. Start timer: ၊ သတ်မှတ်ထားသော အလုပ်ကို ၂၅ မိနစ်အထိ အာရုံစိုက်ပါ
  4. Take a break: ၅ မိနစ် အနားယူပါ
  5. လုပ်ဆောင်ပါ: — ၃ မှ ၅ ကြိမ်ထပ်၍ အလုပ်ကို ပြီးမြောက်အောင်လုပ်ပါ
အခြေခံ လုပ်ဆောင်ချက်များ

  • ကိုယ်ပိုင်ချိန်ညှိမှု: အာရုံစိုက်ချိန်၊ အနားယူချိန်နှင့် သံမြည်အသံများကို ကိုယ်တိုင်ပြင်ဆင်နိုင်သည်။
